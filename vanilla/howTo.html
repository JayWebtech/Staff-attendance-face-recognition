How To Use

Below are instructions on how to use the system and handle any problem you might occur when using it

Registration

The registration stage captures are student's face and saves it to an online database in order to perform face recognition later on in the attendance stage

  • click the register button on the homepage
  • Wait for few seconds as the face recognition initializes
  • The Face recognition system would capture your face the moment it detects it
  • After capturing, a new page for you to enter your name is loaded
  • Input your name and save Note, this is the name that the system would use for all further attendance purpose
  • you are then navigated back to the home page
  • registration Completed

You can't capture your face manually or upload an image to avoid computer vision errors

Attendance

The Attendance stage compares a given face with the images in the database and when a match is found, it triggers a function that adds the current data and time the the student's file(database)

  • click the Attendance button on the homepage
  • Wait for few seconds as the face recognition initializes
  • The Face recognition system would capture your face the moment it detects it
  • After capturing, a comparison is done on the back end and a result is returned
  • After few seconds the page is reloaded
  • Attendance Completed

Alternative

Few seconds can range from 1second to about 15seconds depending on the internet speed, the CPU or GPU capacity and other variables, based on this the Face attendance might not always be optimal

In case of large scale events, an alternate option was included, that takes attendance by just a click of a button.

  • click the Alt Attendance button on the homepage
  • Wait for few seconds the database loads all students
  • After that with just a click on the name the attendance database is updated
  • Attendance Completed
This Alternative also comes with full offline support

Why this Method

What this method lacks in speed, it gains in reach

  • Registration can be done from anywhere
  • A cloud database with all your students that can be accessed anywhere at anytime
  • This system can be built upon to include other features like courses which would then promote online learning
  • The system is highly scalable and other features can be easily integrated like User Auth, Email Notification and lots more
